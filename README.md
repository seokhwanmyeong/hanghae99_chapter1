1. Intro Project

J-GATI는 조깅 파트너를 모집하는 웹 애플리케이션입니다.
혼자하는 운동을 떠나 같이 동료를 모집하여 즐길 수 있으며, 
즐거운 조깅 모임을 만드실 수 있습니다.
현재 단계는 앱의 중심기능인 모집과 기본 서버를 토대로하는 기초 프레임을 만드는 단계이며,
앞으로 더 다양한 기능으로 여러분과 함께 할 예정입니다.
